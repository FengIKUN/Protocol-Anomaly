-- Aris Thorne

你已成功潜入外部网关。但核心数据被 JWT (JSON Web Token) 鉴权系统锁定了。

系统检测到你当前的 Session Role (会话角色) 仅为 guest（访客）。

[REQUIRED_PRIVILEGE]: admin
[TARGET_NODE]: Intranet_Browser -> Admin Dashboard
