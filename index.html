如果你看到了这份文件，说明你已经触发了引线，并即将接入 OBLIVION CORP. 公司的内部网络。

我是 <Null>。接下来的旅程不是普通的解谜游戏。现实与虚拟的边界将在这里消融，你所看到的界面，很多时候只是用来蒙蔽双眼的谎言。

在进入游戏之前，确保你的大脑装备了以下“武器”：

[REQUIRED_ARSENAL] 工具箱清单

[F12] 浏览器开发者工具

按键盘上的 F12（或右键“检查”）。学会在 Elements 中修改页面结构，在 Network 中寻找数据，在 Application 中伪造身份。

[Terminal] 命令行终端模拟

游戏中没有精美的 UI 按钮。遇到不会的命令，记得使用 help。使用 nmap, gobuster 等工具去探测未知领域。

[OSINT] 常识

不要把思维局限在这个网页内。如果遇到结尾带 '=' 的乱码，试着找个 Base64解码器；如果遇到刺耳的音频，试着看看它的频谱图。善用现实世界的神器（如 CyberChef）。

[WARNING] 系统警告

一旦建立连接，系统的反制程序将会启动。保持隐蔽，不要相信 AI 说的每一句话。祝你好运。
